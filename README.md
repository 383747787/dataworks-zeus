##宙斯(zeus)是什么
宙斯是一个完整的Hadoop的作业平台  
从Hadoop任务的调试运行到生产任务的周期调度
宙斯支持任务的整个生命周期

从功能上来说，支持：  
Hadoop MapReduce任务的调试运行  
Hive任务的调试运行  
Shell任务的运行    
Hive元数据的可视化查询与数据预览  
Hadoop任务的自动调度  

Zeus2是在阿里Zeus的基础上开发的支持Hadoop2 的调度平台，Zeus2相关说明在WIKI中可以查看，且文档正在持续完善中
